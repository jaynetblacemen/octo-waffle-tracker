# A simple tracker #
## NOTE: This project is only for education purpose and i dont take resposibilty if it is misused. ##
*This project is heavily inspired by [raz-js tracker](https://github.com/Raz-js/Discord-IP-Logger).*

 A lightweight and simple tracker that send the log instantly to your discord webhook as soon as the website is loaded and redirect to your selected website website.
 The app.js contain the discord logging code and you have to put your discord webhook in that file, at the top at its desired spot, and then host the index.html, when someone visit the html page, it run the app.js and send the detail to discord.
 
 You shall use ublock origin and block the javascript from excuting to keep yourself safe.
